Being Social

The “Being Social” is a social media platform which is used to connect with friends and we can be able to share our memories, ideas and thoughts. This platform supports all major features that a social media must require like Responsiveness, Privacy, Sharing and Messaging. It has some microservices that will used to do the respective process like Authentication, Comments, User services etc., we have created a Responsive Frontend and we have connected the Backend to that Frontend application using React JS. The images and posts we share will store in a platform called “Cloudinary” and user details will be stored in “MongoDB”. By connecting all the Services into a single platform by using some “Routes” we can be able get a clean social media platform “Being Social”..Social media is a web‐based technology to facilitate social interaction between a large group of people through some type of network. In common widely used network is the Internet. But social media platforms are also for local networks as well.Social media is growing rapidly and becoming an inevitable part of everyday life, because of the latest technological revolution. This stunning growth is due to the increasing usage of smart phones like BlackBerrys, Q-Mobile, Androids and iPhones. These Smart phones make it easy to access any social media platform from anywhere virtually. The mobile versions of these social media sites are so easy to access made it user friendly
